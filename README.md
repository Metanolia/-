Создал файл который содержит мою первую консольную программу на C++(после HelloWorld :3))), которая рассчитывает геометрические характеристики шести фигур и работает в цикле до выбора выхода. В начале у меня подключаются стандартные заголовки такие как iostream, iomanip, limits и string. Константа pi определена глобально, чтобы использовать её в формулах для круга.

Функция showMenu() выводит на экран список доступных фигур и команду выхода. Функция readChoice() читает целое число из ввода и повторяет запрос, если пользователь ввёл некорректное значение. В ней используется std::cin.fail() для обнаружения ошибок ввода, затем поток очищается и пропускается вся оставшаяся строка. Аналогично, readPositiveDouble() запрашивает вещественное число с заданным приглашением, проверяет, что ввод корректен и значение положительное, и возвращает его только после успешного ввода.

Далее определены функции вычисления площадей и периметров для каждой фигуры: круг, прямоугольник, треугольник, квадрат, параллелограмм и трапеция. Каждая функция возвращает числовой результат по соответствующей формуле. Для круга есть отдельные функции areaCircle() и circumferenceCircle(), для прямоугольника — areaRectangle() и perimeterRectangle(), для квадрата — areaSquare() и perimeterSquare(), а для остальных фигур — функции площади и периметра по стандартным формулам.

В main() программа входит в бесконечный цикл while (true). В каждой итерации показывается меню, считывается выбор пользователя и выполняется соответствующий блок if / else if. Для каждой фигуры запрашиваются нужные параметры через readPositiveDouble(), затем вычисляется площадь и, где нужно, периметр или длина окружности. Результат выводится в формате с двумя десятичными знаками благодаря std::fixed << std::setprecision(2).

Если пользователь выбирает 7, программа выводит сообщение о выходе и прерывает цикл break. При некорректном выборе выводится сообщение о том, что следует выбрать число от 1 до 7. После каждого расчёта добавляется пустая строка для читабельности, а в конце main() возвращает 0, что означает успешное завершение.
